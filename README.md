# SpendX

**AI-Powered Smart Finance Manager**

A modern personal finance mobile application built with React Native and Expo.

## Project Structure

- **`mobile-app/`** - React Native mobile application
  - See [mobile-app/README.md](./mobile-app/README.md) for setup instructions

## Quick Start

```bash
# Clone the repository
git clone https://github.com/PrathikReddy560/SpendX.git

# Navigate to mobile app
cd SpendX/mobile-app

# Install dependencies
npm install

# Start development server
npx expo start
```

## Tech Stack

- React Native 0.76
- Expo SDK 54
- TypeScript
- React Native Paper (Material Design)
